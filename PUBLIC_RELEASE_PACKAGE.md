## 🚀 Quick Start

### 1. Clone Repository
```bash
git clone https://github.com/backupsonbackupsrobby-cyber/ENGINE2.git
cd ENGINE
```

### 2. Deploy with Docker
```bash
docker-compose -f docker-compose-production.yml up -d
```

### 3. Access Dashboards
```
EHF Performance:     http://localhost:9001
Smart Home Control:  http://localhost:9000
Prometheus:          http://localhost:9090
Grafana:             http://localhost:3000
Jaeger:              http://localhost:16686
AlertManager:        http://localhost:9093
```

### 4. Test APIs
```bash
# EHF Performance
curl http://localhost:9001/api/ehf/status

# Smart Home Devices
curl http://localhost:9000/api/zha/devices

# TRON Sync Status
curl http://localhost:9000/api/tron/status
```
